linux常用命令

命令 作用 使用方式
Man命令 详细了解某命令 ［root @teacher root］# man ls（了解ls命令的详细信息）
pwd 命令 “print working directory”（打印工作目录） [root@teacher apache]# pwd（打印当前位置）
cd 命令 改变工作目录
使用方式 作用
cd~ 去你的登录目录
cd/ 去整个系统的根目录
cd/root 去根用户的主目录;你必须是根用户才能访问该目录
cd/home 去home目录，用户的登录目录通常贮存在此处
cd.. 返回上一级
cd/文件夹1/文件夹2 去文件夹2
cd../../文件夹1/文件夹2 返回上两级后去文件夹2
ls 命令 显示当前内容
  1. -a — 全部（all）。列举目录中的全部文件，包括隐藏文件（.filename）。位于这个列表的起首处的 .. 和 . 依 次是指父目录和你的当前目录
  2. -l — 长（long）。列举目录内容的细节，包括权限（模式）、所有者、组群、大小、创建日期、文件是否是到 系统其它地方的链接，以及链接的指向。
  3. -F — 文件类型（File type）。在每一个列举项目之后添加一个符号。这些符号包括：/ 表明是一个目录；@ 表 明是到其它文件的符号链接；* 表明是一个可执行文件。
  4. -r — 逆向（reverse）。从后向前地列举目录中的内容。
  5. -R — 递归（recursive）。该选项递归地列举所有目录（在当前目录之下）的内容。
  6. -S — 大小（size）。按文件大小排序
locate 命令 搜寻文件或目录 ocate finger(搜索文件或目录带finger的)
clear 命令 清除终端窗口 clear
cat 命令 示文件的内容（经常和more搭配使用），或者是将多个文件合并成一个文件。 -----
head 命令 查看文件的开头部分 head -20 <文件名> （阅读文件前20行）
tail 命令 查看文件结尾的十行 tail -20 <文件名> （阅读文件后20行）
grep 命令 文件中查找指定的字串 grep coffee 11.txt （在11.txt中查找coffee）
chmod 命令 改变文件或目录的访问权限 chmod o+w sneakers.txt （给其它人写入文件 sneakers.txt 的权限）
cp 命令 cp(copy)命令可以将文件或目录复制到其他目录 cp <源> <目标>
mv命令 移动文件
  1. -i — 移动。
  2. -f — 强制。移动文件有问题不报错。
  3. -v — 详细。显示文件的移动进度。
  4. mv 111.txt /home/newuser/sneakers.txt /home/newuser/tigger（移动所在目录的111.txt到tigger）
mkdir 命令 建立目录 [root@teacher tmp]#mkdir datal（在系统中建立datal 子目录）
rm 命令 删除目录中的文件或目录本身，对于链接文件，只是删除了该链接，原有文件保持不变。 rm 111.txt （删除111.txt）
