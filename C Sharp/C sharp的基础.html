C#的基础

  1. 变量名命名规则与C相同（可用中文命名变量，但可能出现奇怪的编译错误或警告）
  2. 变量定义规则与C相同
  3. 可以利用var（只编译第一次的）/dynamic（每次编译都判断）自动识别变量定义类型
  4. 基础类型，变量，函数，运算皆与C一样
  5. 输出与输入函数与C不一样
  6. if条件语句与C相同
  7. switch判断能用字符（中文）
  8. while/do...while/for循环语句与C相同
  9. continue和break使用方式相同
  10. 函数传递变量时，定义变量前加ref，能使变量在函数内修改后，复制回函数外（偏向于地址）
  11. 自己定义对象class: class 类型名字 {/*定义*/}，例：
    • class Human{ int age; string name; }
    • 读写：human.name = "Name"
    • 权限：定义时某变量前加public表示能在类外读写，加private则只能在类内写；类外能读
    • 类内可以定义静态函数和静态变量（static）即文件外不能调用，可以利用类内函数修改类内变量值
    • 类内可以定义const常量
  12. Struct结构体与C相同（偏向于函数值）
    • 如果需要将Struct的引用传递给函数内部，需要使用关键词ref
  13. 继承：Class B：A{/**/}
    • A叫做父类，B叫做子类
    • B从A中继承：属性和方法
    • 权限：定义时某变量前加public表示能在类外读写，加在private则只能在类内读写，加protected只能自己和子类读写
    • 继承时在同一个函数名前加override覆盖父类同一函数，表示给子类实现
    • 父类变量可以存储子类变量（地址），不能反过来
    • 有抽象函数时，继承时需加abstract，即父类不用实现但子类有该函数，父类需加上该函数定义
  14. 泛型：泛用结构体/类，可设置任意类型参数等
  15. 容器
    • Array定义：固定长度数组
    • List定义：动态数组，自动增长
    • Dictionary：使用key获取value
    • HashSet：保证数组元素唯一，插入使用，若数组有相同值则插入失败
    • Queue定义：数组动态增长，先进先出
    • stack：数组动态增长，后进先出
  16. 函数指针
  17. 反射
  18. 宏：与C语言相似
    命令 作用
    #define 定义宏
    #undef 取消宏定义
    #if 判定是否定义了对应的宏
    #else 如果判断没有则xxxx
    #endif 结束标记
  19. 异常
